# calculator-on-java
・使用方法について
  
	一般の電卓と同じように作っております。
  	GUI化しておりますので操作には困らないと思います。
	数字を入力し演算記号をクリックしたあと、もう一つ数字を入力してから＝をクリックすると結果が出てきます。
	括弧による計算順序の変更はできません。順序にはお気を付けください。
